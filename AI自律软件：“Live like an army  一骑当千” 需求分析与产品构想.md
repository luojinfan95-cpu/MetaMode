# AI自律软件：“Live like an army / 一骑当千” 需求分析与产品构想

## 背景

时代在发展，技术在进步，如今的ai革命可比肩当初的互联网革命，但还没有人找到破解之道：互联网革命已经吞噬了大多数人大多数的专注力和时间，而当下用ai开发应用虽越来越快，但能夺取到的注意力却越来越少。市场的需求似乎只有那么多，而且已经被消耗殆尽。但这其实隐含了一个假设：“专注力就代表需求，在软件上投入的专注力越多即使用这个软件越多，即软件产品做得越好。”

这种假设反过来说成不成立？在软件上投入的专注力越少，软件做得越好？有一个这样的类别：番茄钟软件，即自我管理。互联网对人专注力的夺取已经超过了合适值，很多人正在开始主动夺回专注力，而这种反其道而行之的软件就是他们的需求。

此外，还有另一个需求点：ai的出现，使互联网变得越来越不真实，屏幕的另一端正在做图灵测试的可能性正在变得越来越大。因此，高质量度过线下时间、减少线上时间的含金量还在升高。大家很快就会发现整个互联网正在变成一个ai占比越来越高的混沌体，上网的性质变成了纯粹的自娱自乐。

## 产品原型构想

建立在以上需求分析的基础上，我们得知，我们想要做的一款产品是“AI+自我管理”，帮助人们提升线下生活的质量。

1、MVP核心功能：

（1）任务拆解：输入自己想完成的任务，ai帮助拆解为可执行的步骤以及执行所需具体时间，基于“知识库1”中提出的各种原理帮助用户无痛顺利执行计划。

（2）习惯养成：输入想要养成的习惯/想要达成的长期目标，ai帮助拆解为一个个小的“微习惯”，基于“知识库1”中提出的国策树形式实现习惯积累，撬动宏观行为模式。

2、必须特征：软件轻量、画风简约，降低启动成本；根据用户数据进行个性化输出，提高用户粘性。

3、锦上添花（选择性必修）：插入游戏化元素，例如自己有一个虚拟形象，这个形象随着自己任务的完成而变得丰满。

注意：“游戏化”≠“玩游戏”，更不等于浪费时间！他只是一个世界观和画风，奠定整个应用的基调和故事，是一个好软件不可或缺的一环
